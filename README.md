# Weather Dashboard

A beginner-friendly weather app that shows live weather by city using the OpenWeather API.

## Features
- Search weather by city
- Shows temperature, humidity, wind, and condition
- Error message for invalid cities

## Tech Stack
- HTML, CSS, JavaScript
- OpenWeather API

## Run Locally
1. Download or clone this project
2. Create `config.js` and add your API key:
   `window.WEATHER_API_KEY = "YOUR_KEY";`
3. Open `index.html` in your browser
